# Sway-dots (for Fedora Sway Spin)

These are my dotfiles for Fedora's Sway Spin, complete with install scripts.

# Required Packages
## Should be installed by default
```
sway
foot
waybar
rofi
```
## Installed by setup-dots
```
wlogout
neovim
```

## Installed by install-full-environment
```
flatpak
flatseal
spotify
spicetify
vs code 
dsicord
brave
gimp
darktable
```
